Help

1. Usage instructions

- Download MusicXML files into your local devices.
- Add files to the app using button.
- Choose your favorite score and start playing.

2. Play modes

2.1. Practice mode

You need a MIDI input device to use this mode. When correct notes are played, the green cursor will advance forward.
Try to catch up with the tempo cursor (the small gray one) while you are on it.

2.2. Listen mode

This mode plays all the score automatically. Make sure you have disabled your device's silent mode and tuned up the volume.

3. Configurations

3.1. Virtual Keyboard

This option toggles the virtual keyboard on the play screen. GREEN keys are for the left hand, BLUE keys are for the right hand.

3.2. Tempo Speed

This speed is relative with the score's tempo, varies from 30% to 180% of the tempo in the current playing notes.

3.3. Measure Range

This option limits range of music measures for playing.

3.4. Measure Repeat

This option allows to play a range of measures repeatedly.

3.5. Staff Toggle

Use this option to practice hands separately.

4. Support

Visit https://help.musetrainer.com for questions and support.
